Arduino Powered Auto-watering System for my mini-Coffee Tree!

Components:
1. Arduino Uno R4 Wifi
2. Generic OLED Screen (I use a SSD1306)
3. Temperature / Humidity Sensor (I use a Grove Temp Hum Sensor Pro v1.3)
4. Capactive Soil Sensor
5. (eventually) a small water pump + other pump engaging components
6. Splunk on Ubuntu
